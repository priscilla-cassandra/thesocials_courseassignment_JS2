# thesocials_courseassignment_JS2
The Socials is a social media application where users can create and share their own posts as well as view posts from other users. You can also follow your favorite users, to make sure you dont miss any of their posts!

## Built with
- Vanilla JavaScript
- HTML
- CSS

## Running the project
1. Make sure you have VS Code installed on your computer
2. Inside VS Code, press Ctrl+Shift+P to open the command pallete
3. Type Git: Clone
4. Paste the URL of the repository: https://github.com/priscilla-cassandra/thesocials_courseassignment_JS2.git
5. Select a folder on your local computer to save the repository
6. When prompted click 'Open' to open the cloned repository in VS Code

How to open the app locally:
1. If LiveServer is installed, right click index.html and click 'open with LiveServer' (The URL will look like this: http://127.0.0.1:5500/index.html)
2. Open from the terminal by typing:
   - Windows: start index.html
   - macOS: open index.html
   - Linux: xdg-open index.html

## Features
- Loginform that allows users to log in to their personal profile
- Registerform that lets you register a new user
- Logged in users can create personal posts (and edit or delete them)
- Logged in users can follow other users
- Logged in users can view other profiles, and all posts from the feed
